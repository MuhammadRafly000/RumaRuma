# Cover Image — Gumroad

Save your cover here.

## Spesifikasi Gumroad

| Aspek           | Nilai                                                     |
| --------------- | --------------------------------------------------------- |
| **Dimensi**     | 1280 × 720 px (16:9) — wajib                              |
| **Format**      | JPG (lebih kecil) atau PNG                                |
| **Ukuran file** | < 1 MB ideal, max 5 MB                                    |
| **Thumbnail**   | Gumroad auto-crop ke kotak 600 × 600 — pastikan elemen penting di tengah |

## Komposisi yang convert tinggi

1. **Mockup browser / device** menampilkan homepage hero
2. **Headline overlay** seperti "Premium React E-Commerce Template" di
   pojok atas/kiri
3. **Subline** kecil seperti "React · Vite · Tailwind · Framer Motion"
4. **Brand badge** (logo kecil) di pojok bawah
5. **Background**: warna brand utama (sage / cream) atau gradient halus

## Tool gratis untuk bikin cover

- **Figma** — template e-commerce cover tersedia di Community
- **Canva** — search "Gumroad cover" atau "product cover"
- **Photopea** — Photoshop-like, gratis, di browser
- **Screen Studio** (macOS, $89) — bisa export still frame jadi cover

## Nama file disarankan

```
cover-1280x720.jpg
cover-thumbnail-600x600.jpg   ← optional override thumbnail
```

Setelah upload ke Gumroad, kamu bisa hapus file dari folder ini.
