# Marketing Folder

Bahan-bahan untuk listing Gumroad. **JANGAN ikut di-zip** ke buyer
package — folder ini hanya untuk kamu (penjual).

```
marketing/
├── README.md              ← file ini
├── GUMROAD-LISTING.md     ← semua copywriting Gumroad
├── cover/                 ← cover image untuk thumbnail Gumroad
│   └── README.md
├── screenshots/           ← gallery screenshot (5–8 buah)
│   └── README.md
└── video/                 ← preview video Loom / Screen Studio
    └── README.md
```

## Workflow

1. Buka `GUMROAD-LISTING.md` — copy bagian yang dibutuhkan ke Gumroad.
2. Bikin cover image (1280×720) → drop ke `cover/`.
3. Screenshot 5–8 halaman utama → simpan di `screenshots/`.
4. Record video preview 30–60 detik → upload ke Loom / YouTube → tempel
   link-nya ke listing.

## Sebelum upload ZIP ke Gumroad

Pastikan folder ini **TIDAK ikut** ke dalam ZIP yang dikirim ke buyer.
Lihat `../PACKAGING.md` untuk script bikin ZIP otomatis yang sudah skip
folder ini.
