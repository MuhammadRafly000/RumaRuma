# Preview Video — Gumroad

Drop your raw video file di sini sebelum upload ke Loom / YouTube.

## Spec video preview

| Aspek      | Nilai                                |
| ---------- | ------------------------------------ |
| **Durasi** | 30–60 detik (ideal), max 90 detik    |
| **Format** | MP4 (H.264) atau MOV                 |
| **Resolusi** | 1920 × 1080 (1080p) minimum        |
| **FPS**    | 30 atau 60                           |
| **Audio**  | Optional — tapi background music halus + voiceover boost konversi |

## Script suggestion (40–50 detik)

```
[0:00 – 0:05] Logo intro + tagline "Premium React E-Commerce Template"
[0:05 – 0:15] Scroll homepage — hero carousel jalan, animasi muncul
[0:15 – 0:22] Klik kategori → grid produk muncul → filter sidebar
              (click brand checkbox, harga range)
[0:22 – 0:30] Klik produk → gallery + info → tambah ke cart
              → cart sidebar slide in
[0:30 – 0:38] Switch ke mobile view → swipe homepage → buka mobile menu
[0:38 – 0:45] Quick flash admin dashboard → checkout flow
[0:45 – 0:50] End card: "RumaRuma — get it on Gumroad" + link
```

## Upload ke

- **Loom** — paling cepat, langsung dapat link, autoplay di Gumroad page
- **YouTube unlisted** — bagus untuk SEO bonus
- **Vimeo** — premium look, paid

## Tempelkan ke Gumroad

Setelah upload, copy link → di Gumroad listing → field "Cover content"
→ pilih "Video URL" → paste. Gumroad akan auto-embed.
