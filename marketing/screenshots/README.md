# Screenshot Gallery — Gumroad

Drop 5–8 screenshot terbaik di folder ini.

## Yang wajib di-capture

1. **Homepage** — hero carousel terlihat penuh
2. **Category page** — filter sidebar + grid produk + sort dropdown
3. **Product detail** — gallery + info panel + tab description
4. **Cart drawer** — sidebar terbuka dengan beberapa item
5. **Checkout** — form lengkap dengan summary aside
6. **Mobile view** — homepage di iPhone frame (penting!)
7. **Admin dashboard** — bukti ada admin panel
8. **404 page** — bonus, menunjukkan attention to detail

## Cara screenshot yang bagus

### Lewat browser

1. Build production: `npm run build && npm run preview`
2. Buka di Chrome → DevTools (F12) → Device Toolbar (Ctrl+Shift+M)
3. Pilih device "Responsive" dan set 1440×900 untuk desktop screenshots
4. Switch ke "iPhone 14 Pro" untuk mobile screenshots
5. Cmd+Shift+P (Mac) / Ctrl+Shift+P (Win) → "Capture full size screenshot"

### Lewat tool (lebih clean)

- **Screen Studio** (macOS) — clean mockup, $89
- **Cleanshot X** (macOS) — $29 one-time
- **Shottr** (macOS) — gratis
- **ShareX** (Windows) — gratis, powerful

## Spesifikasi file

| Aspek      | Desktop       | Mobile        |
| ---------- | ------------- | ------------- |
| **Resolusi** | 1440 × 900    | 390 × 844     |
| **Format**   | PNG (sharp) atau JPG quality 90 |             |
| **Ukuran**   | < 500 KB per file (compress di tinypng.com) |    |
